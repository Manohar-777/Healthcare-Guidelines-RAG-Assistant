# CDC Guideline: Isolation_And_Return_To_Work (2023)

## Overview
CDC guidance for isolation and return to work.

## Principles
Adapt to local policy. Perform risk assessment. Non-prescriptive baseline.

## Recommendations
- symptomatic staff should not work until criteria met.
- use time or test based strategies.
- document return to work clearance.

## Implementation
Train staff; monitor adherence; document deviations; quarterly review.
